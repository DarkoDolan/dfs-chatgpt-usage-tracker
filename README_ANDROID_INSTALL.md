# DFS ChatGPT Usage Tracker - Android PWA

A simple installable web app for Android.

## What it does
- Manually tracks ChatGPT messages
- Manually tracks image uploads and file uploads
- Estimates uploaded storage from the file sizes you enter
- Starts/resets a 3-hour usage window
- Exports backup JSON and CSV
- Works offline after install

## What it cannot do
It cannot automatically read your hidden ChatGPT usage counters. It is a manual helper.

## Install on Samsung / Android
1. Host this folder somewhere HTTPS, ideally GitHub Pages.
2. Open the hosted index.html link in Chrome on your phone.
3. Tap the three dots menu.
4. Tap Add to home screen.
5. If Chrome shows Install, tap Install.

## Quick test without hosting
You can open index.html directly in a browser, but proper app install/offline mode needs HTTPS hosting.
