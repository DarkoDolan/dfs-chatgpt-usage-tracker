# DFS ChatGPT Usage Tracker

A small manual usage tracker for ChatGPT sessions, made for **Dolan Friendly Services**.

Live app:

<https://darkodolan.github.io/dfs-chatgpt-usage-tracker/>

## What it does

- Tracks message count manually
- Tracks image uploads manually
- Tracks file uploads manually
- Estimates storage from file sizes you enter
- Keeps data locally in the browser/app
- Exports backup JSON and CSV logs
- Installs on desktop Chrome and Android Chrome as a small web app

## Important

This app **does not automatically read ChatGPT limits**. It is a private/manual helper so you can tap along and avoid surprise limits.

## How to install on PC

1. Open the live app in Chrome.
2. Click the install icon in the address bar, or open Chrome menu.
3. Choose **Install page as app**.

## How to install on Android

1. Open the live app in Chrome on Android.
2. Tap the three-dot menu.
3. Tap **Add to Home screen** or **Install app**.
